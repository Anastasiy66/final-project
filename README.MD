# Статистический учет несчатных случаев

## Описание проекта
Настольное Python-приложение для учета несчастных случаев на производстве с анализом и визуализацией данных

## Функционал
- Подсчет  легких и тяжелых несчастных случаев;
- Хранение данных - в CSV;
- Анализ пострадавших по типу несчастных случаев;
- Визуализация данных;
- GUI Tk.

## Запуск
```
pip install -r requirements.txt
python main.app
```
## Инструменты
-pandas
-matplotlib
